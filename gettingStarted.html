Getting started

Dependencies

Core

The core module includes the DSLs that allow you to create the checkpoint stages to instrument your Akka Streams pipeline.

sbt
libraryDependencies += "com.github.svezfaz" %% "akka-stream-checkpoint-core" % "0.0.2-SNAPSHOT"
Maven
<dependency>
  <groupId>com.github.svezfaz</groupId>
  <artifactId>akka-stream-checkpoint-core_2.12</artifactId>
  <version>0.0.2-SNAPSHOT</version>
</dependency>
Gradle
dependencies {
  compile group: "com.github.svezfaz", name: "akka-stream-checkpoint-core_2.12", version: "0.0.2-SNAPSHOT"
}

scaladsl and javadsl

There are two separate packages named akka.stream.checkpoint.scaladsl and akka.stream.checkpoint.javadsl with the API for Scala and Java. These packages contain a Checkpoint class with factory methods to create a checkpoint given a name (label) and a backend.

Examples

Dropwizard

These examples require akka-stream-checkpoint-core and akka-stream-checkpoint-dropwizard modules.

Scala
implicit val metricRegistry = new MetricRegistry()

import akka.stream.checkpoint.DropwizardBackend._

Source(1 to 100)
  .via(Checkpoint("produced"))
  .filter((x: Int) ⇒ x % 2 == 0)
  .via(Checkpoint("filtered"))
  .runForeach(println)
Java
final MetricRegistry metricRegistry = new MetricRegistry();
final CheckpointBackend backend     = DropwizardBackend.fromRegistry(metricRegistry);

Source.range(1, 100)
    .via(Checkpoint.create("produced", backend))
    .filter(x -> x % 2 == 0)
    .via(Checkpoint.create("filtered", backend))
    .runForeach(System.out::println, materializer);

Kamon

These examples require akka-stream-checkpoint-core and akka-stream-checkpoint-kamon modules.

Scala
import akka.stream.checkpoint.KamonBackend._

Source(1 to 100)
  .via(Checkpoint("produced"))
  .filter((x: Int) ⇒ x % 2 == 0)
  .via(Checkpoint("filtered"))
  .runForeach(println)
Java
final CheckpointBackend backend     = KamonBackend.instance();

Source.range(1, 100)
    .via(Checkpoint.create("produced", backend))
    .filter(x -> x % 2 == 0)
    .via(Checkpoint.create("filtered", backend))
    .runForeach(System.out::println, materializer);

Sugared DSL

Regardless of the chosen backend, some syntactic sugar is available for Scala users under scaladsl.Implicits.

Scala
import akka.stream.checkpoint.DropwizardBackend._
import akka.stream.checkpoint.scaladsl.Implicits._

Source(1 to 100)
  .checkpoint("produced")
  .filter((x: Int) ⇒ x % 2 == 0)
  .checkpoint("filtered")
  .runForeach(println)
