Backends

Every checkpoint needs a backend to store metrics readings.

Dropwizard

The Dropwizard backend creates the following metrics for each checkpoint. Note that each metric will be prefixed with the checkpoint label you specify.

  • {label}_push_latency: histogram to store the latencies between push and pull signals in nanoseconds
  • {label}_pull_latency: histogram to store the latencies between pull and push signals in nanoseconds
  • {label}_throughput: meter to store the throughput rate of the checkpoint
  • {label}_backpressure_ratio: histogram to store the percentage of time the checkpoint is backpressuring
  • {label}_backpressured: counter to store the current backpressure state of the checkpoint

Kamon

The Kamon backend creates the following metrics for each checkpoint. Note that each metric will be prefixed with the checkpoint label you specify.

  • {label}_push_latency: histogram to store the latencies between push and pull signals in nanoseconds
  • {label}_pull_latency: histogram to store the latencies between pull and push signals in nanoseconds
  • {label}_throughput: counter to store the throughput rate of the checkpoint
  • {label}_backpressure_ratio: histogram to store the percentage of time the checkpoint is backpressuring
  • {label}_backpressured: gauge to store the current backpressure state of the checkpoint

Custom

Custom backends can be created by implementing CheckpointBackend and CheckpointRepository interfaces.

Scala
implicit val loggingBackend: CheckpointBackend = new CheckpointBackend {
  override def createRepository(name: String): CheckpointRepository = new CheckpointRepository {

    override def markPush(latencyNanos: Long, backpressureRatio: Long): Unit =
      println(s"PUSH - $name: latency:$latencyNanos, backpressure ratio:$backpressureRatio")

    override def markPull(latencyNanos: Long): Unit =
      println(s"PULL - $name: latency:$latencyNanos")
  }
}

Source(1 to 100)
  .via(Checkpoint("produced"))
  .filter((x: Int) ⇒ x % 2 == 0)
  .via(Checkpoint("filtered"))
  .runForeach(println)
Java
final CheckpointBackend backend = new CheckpointBackend() {
    @Override
    public CheckpointRepository createRepository(String name) {
        return new CheckpointRepository() {
            @Override
            public void markPush(long latencyNanos, long backpressureRatio) {
                System.out.println(String.format("PUSH - %s: latency:%d, backpressure ratio: %d",
                        name, latencyNanos, backpressureRatio));
            }

            @Override
            public void markPull(long latencyNanos) {
                System.out.println(String.format("PULL - %s: latency:%d",
                        name, latencyNanos));
            }
        };
    }
};

Source.range(1, 100)
    .via(Checkpoint.create("produced", backend))
    .filter(x -> x % 2 == 0)
    .via(Checkpoint.create("filtered", backend))
    .runForeach(System.out::println, materializer);
